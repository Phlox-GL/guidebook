# Color

Notice that Pixi.js takes colors in hex numbers.

```cirru
{} $ :color 0xaaaaff
```

For HSL:

```cirru
phlox.core/hslx 200 80 80
```

For HSLuv, based on [hsluv](https://www.npmjs.com/package/hsluv):

```cirru
phlox.core/hsluvx 200 80 80
```

For HCL:

```cirru
phlox.core/hclx 200 80 80
```
